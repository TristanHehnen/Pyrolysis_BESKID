# Data Processing

## Experimental Data

The experimental data is processed by the `Prepare*Experiments.ipynb` notebooks. Once you've cloned this repo to your machine and added the experimental data to the respective directory, run the notebooks to prepare the experimental data to create the respective files. The results will be saved in `*_Output` directories that are created by the notebooks.

The processing of the experimental data comprises computing averages and standard deviations of the various data series. The resulting data is used as targets for the inverse modelling processes and to assess the model performance.
