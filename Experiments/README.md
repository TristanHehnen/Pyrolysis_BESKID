# Experimental Data

The experimental data used used for this work is processed using the Jupyter scripts in the `DataProcessing` directory.

The raw experimental data is located in the following Git repository: https://github.com/FireDynamics/BESKID_ExperimentalData

Please copy/clone the content of the above repo here, i.e. into `BESKID_ExperimentalData`, such that the data processing scripts can work properly.
